一、购物车
    1.1 添加购物车：http://cg.liaidi.com/?m=api&appid=应用appid&token=API令牌&c=cart&a=add
    参数1：itemid=商品ID
    参数2：quantity=产品数量

    1.2 删除购物车：http://cg.liaidi.com/?m=api&appid=应用appid&token=API令牌&c=cart&a=delete
    参数1：itemid=商品ID

    1.3 更新购物车：http://cg.liaidi.com/?m=api&appid=应用appid&token=API令牌&c=cart&a=update
    参数1：itemid=商品ID
    参数2：item[字段名]=字段值

    1.4 获取购物车商品列表：http://cg.liaidi.com/?m=api&appid=应用appid&token=API令牌&c=cart&a=get_item_list

    1.5 获取结算商品列表：http://cg.liaidi.com/?m=api&appid=应用appid&token=API令牌&c=cart&a=get_settle_items

    1.6 结算商品：http://cg.liaidi.com/?m=api&appid=应用appid&token=API令牌&c=cart&a=settlement
    请求方式：POST
    参数1：uid=用户ID
    参数2：orders=订单数据 JSON格式，格式如下
    [
        {
            shop_id:店铺ID，
            shop_name:店铺名称
            pay_type:付款方式，1=余额支付，2=在线支付，3=货到付款
            shipping_type:配送方式，1=快递，2=物流，3=自取或无物流
            remark:订单备注
            itemids:[1,2,3,4] 订单商品ID
        },
        {
            shop_id:店铺ID，
            shop_name:店铺名称
            pay_type:付款方式，1=余额支付，2=在线支付，3=货到付款
            shipping_type:配送方式，1=快递，2=物流，3=自取或无物流
            remark:订单备注
            itemids:[1,2,3,4] 订单商品ID
        }
        .......
    ]
    参数3：address_id=收货地址ID
